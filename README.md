# ConectaEdu

## Como rodar o site

1. Clone o repositório:
git clone https://github.com/joaofiedler/ConectaEdu.git

2. Entre na pasta do projeto:
cd ConectaEdu

3. Instale as dependências:
npm install

4. Inicie o servidor:
npm start

5. Abra o navegador e acesse:
http://localhost:3000

O site estará rodando localmente.
